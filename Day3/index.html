Explain inline , internal and external stylesheet

There are three way to add a CSS styling in a page we can use: 

    Inline: direct styling inside a tag. this is the highest priority among three

    Internal: styling inside a head tag. we use directly style the page inside style elemnts in head.

    External : It is most commonly used way of linking a CSS file to a HTML page. It is more convinient to have CSS code in a seperate file so we can make changes quickly.

    Padding property lets us provide value to set the amount of padding we need

What is box sizing?
 


The box-sizing property defines how the width and height of an element are calculated: should they include padding and borders, or not.



it takes values like: 
    border-box: it includes the border and padding in the a box sizing.





    content-box: it excludes the border and padding in the a box sizing. and only considers the content area width and height.





    initial: Sets this property to its default value.





What is animation delay?
 


    In CSS animation property, animation delay is a sub-property which enables us to add a certain delay before the animation or change starts in the element.

    for example if we give delay of 200ms than we will see the animation starting after 200 milliseconds delay only.






Which property you will use to merge cells vertically in grid?
 



    if we want to merge vertically we will use property like grid-column-star and grid-column-end or for short-hand property we use grid-column




Which property you will use to merge cells horizontally in grid?
